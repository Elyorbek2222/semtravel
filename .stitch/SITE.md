# SEM Travel — Site Documentation

## 1. Vision
Premium travel agency landing page targeting Uzbek travelers. Addresses core fears: uncertainty, being stranded, currency fluctuations. Positions SEM Travel as the trusted 24/7 partner with price guarantees in UZS.

## 2. Brand
- Name: SEM Travel
- Logo: Starfish icon (teal/orange) + "SEM Travel" wordmark
- Colors: #1B3A6B (deep blue, trust), #C9A84C (gold, premium), #FFFFFF (white)
- Fonts: Playfair Display (headlines), Inter (body)
- Tone: Warm, trustworthy, premium

## 3. Target Audience
Uzbek travelers (Tashkent) who fear: airport abandonment, price changes (USD→UZS), visa complications, travel uncertainty.
