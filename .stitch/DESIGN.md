# SEM Travel — Design System

## 1. Color Palette
- Primary Blue: #1B3A6B (trust, navigation, buttons)
- Gold Accent: #C9A84C (premium, CTAs, borders)
- White: #FFFFFF (background)
- Light Gray: #F8F9FA (section backgrounds)
- Text Dark: #1A1A2E (headings)
- Text Body: #4A4A68 (body text)
- Text Muted: #8B8BA7 (captions, secondary)

## 2. Typography
- Headlines: Playfair Display, serif, weights 400/700
- Body: Inter, sans-serif, weights 400/500/600
- Hero H1: 56px / 64px line-height (desktop), 36px (mobile)
- Section H2: 40px (desktop), 28px (mobile)
- Card H3: 22px
- Body: 16px / 24px line-height

## 3. Spacing
- Section padding: 96px vertical (desktop), 64px (mobile)
- Container max-width: 1200px, padding: 0 24px
- Card gap: 32px
- Component gap: 24px

## 4. Components
- Primary Button: bg #C9A84C, text white, rounded-full, px-8 py-4, hover darken
- Secondary Button: border #1B3A6B, text #1B3A6B, rounded-full, px-8 py-4
- Cards: white bg, shadow-lg, rounded-2xl, gold border-top 4px
- Nav: white bg, shadow-sm, sticky top

## 5. Motion
- Transitions: 300ms ease
- Hover: translateY(-4px) on cards
- Smooth scroll on anchor links

## 6. Design System Notes for Stitch Generation
Light minimalist premium travel agency. White background #FFFFFF. Deep blue #1B3A6B for trust elements. Gold #C9A84C for premium accents. Playfair Display serif headlines, Inter body. Cards with gold top border. Full-width hero with dark overlay on travel photography. Russian language content. Mobile-first responsive.
